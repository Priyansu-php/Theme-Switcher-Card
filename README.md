# 🌗 Theme Switcher using React Context API

A simple and responsive **Theme Switcher** built with **React**, **Context API**, and **Tailwind CSS v4**. This project demonstrates how to manage global state using Context and implement a light/dark mode toggle across the application.

## 🚀 Features

- 🌞 Light & 🌙 Dark mode switching
- ⚛️ Global state management using React Context API
- 🎯 Custom `useTheme` hook
- 🎨 Responsive UI with Tailwind CSS v4
- 💫 Smooth hover animations and transitions
- 👤 Personalized profile card
- 🔗 GitHub & LinkedIn links

## 🛠️ Tech Stack

- React
- Context API
- Tailwind CSS v4
- Vite
